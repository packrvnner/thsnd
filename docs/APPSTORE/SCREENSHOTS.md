# Screenshot Plan — 5 shots, one story

Required device sizes: 6.9" (1320×2868) and 6.5" (1284×2778 or 1242×2688). Take in Simulator (iPhone 16 Pro Max + iPhone 11 Pro Max), `Cmd+S` saves PNGs at native resolution. No device frames, background stays the app's own black. Caption text overlays in Inter 600, white, upper-left, small — the UI is the poster.

| # | Screen | Setup | Caption |
|---|---|---|---|
| 1 | Markets | Live data loaded, epoch countdown visible | `Every number is on-chain.` |
| 2 | Vault | Watch the treasury Safe so a position renders; slider at 4y | `Lock. vTHSND = amount × time.` |
| 3 | Burn | Chart with burn history visible | `Supply only goes down.` |
| 4 | Tiers | Tier table loaded, T1000 row visible | `T1000: fees declined to exist.` |
| 5 | Company | Milli card + addresses list | `Verified addresses. One tap to the explorer.` |

Rules: no simulated balances that don't exist on-chain (reviewers cross-check), no profit numbers in captions, `SYS:` lines visible where they naturally appear. Same five shots for both size classes.
